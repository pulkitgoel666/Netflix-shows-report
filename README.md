# Netflix-shows-report
This repository contains a dynamic dashboard created using Power BI to visualize and analyze the Netflix titles dataset. The project aims to provide insights into the streaming service's content, highlighting trends and key metrics.

Key Features:

    Dataset: Utilizes the Netflix Titles dataset, which includes comprehensive information about various titles available on the platform, including genres, release years, ratings, and duration.

    Data Cleaning: The dataset was meticulously cleaned and prepared using Excel to ensure data accuracy and consistency. This involved:
        Removing duplicates
        Handling missing values
        Standardizing formats for analysis

    Power BI Dashboard: The dashboard displays:
        Visual insights on title popularity and viewership
        Genre distribution
        Trends over time
        Ratings analysis and comparisons

    Interactive Elements: Users can interact with the dashboard to filter data by genres, years, and ratings, providing a tailored experience.

Getting Started:

    Clone the repository to your local machine.
    Open the Power BI file to explore the dashboard and data visualizations.
